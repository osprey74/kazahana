# Release Notes — v1.4.7

## New Features

- **Pinned post display on profile** — When viewing a user's profile, their pinned post is now displayed at the top of the Posts tab with a pin icon and "Pinned post" label. This uses the Bluesky API's `includePins` parameter and `reasonPin` indicator, matching the behavior of the official Bluesky app. Supported in all 11 languages.

# リリースノート — v1.4.7

## 新機能

- **プロフィールのピン留め投稿表示** — ユーザーのプロフィール画面を表示した際、そのユーザーがピン留めしている投稿が「投稿」タブの一番上にピンアイコンと「ピン留め投稿」ラベル付きで表示されるようになりました。Bluesky APIの `includePins` パラメータと `reasonPin` インジケータを使用しており、公式Blueskyアプリと同等の動作です。全11言語に対応しています。
