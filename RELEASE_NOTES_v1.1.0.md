# kazahana v1.1.0

## What's New

### Added

- **Quote post creation** — Create quote posts with `app.bsky.embed.record`
- **CI/CD with GitHub Actions** — Automated cross-platform builds (Windows x64, macOS Intel, macOS Apple Silicon) on version tag push
- **macOS support** — macOS binaries now available (Intel and Apple Silicon)
- **Installation guides** — Localized installation guides for all 11 languages on the official website

### Fixed

- **Profile header** — Removed misplaced logout button from profile header

# kazahana v1.1.0

## 新機能

### 追加

  **引用投稿の作成** — `app.bsky.embed.record` による引用投稿の作成
  **GitHub Actions CI/CD** — バージョンタグプッシュ時の自動クロスプラットフォームビルド（Windows x64、macOS Intel、macOS Apple Silicon）

  **インストールガイド** — 公式サイトに全11言語のインストールガイドを追加
