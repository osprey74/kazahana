# kazahana v1.4.2

## What's New

### Added

- **Window state persistence** — Window size and position are now saved on exit and restored on the next launch. Powered by `tauri-plugin-window-state`.
- **Improved system tray** — Left-click the tray icon to restore and focus the window. Right-click menu now provides three actions: "Open Window," "Minimize," and "Exit."
- **Close button behavior setting** — A new option in Settings lets you choose what the close (✕) button does: "Exit the application" (default) or "Minimize to system tray." When minimize is selected, a helpful note explains how to fully quit via the tray icon.

# kazahana v1.4.2

## 新機能

### 追加

- **ウィンドウサイズ・位置の保存と復元** — 終了時にウィンドウのサイズと画面上の配置位置を保存し、次回起動時に復元します。`tauri-plugin-window-state` を使用。
- **タスクトレイ操作の改善** — タスクトレイアイコンを左クリックするとウィンドウを復元・最前面に配置します。右クリックメニューに「Open Window」「Minimize」「Exit」の3項目を提供。
- **閉じるボタンの動作設定** — 設定画面に閉じる（✕）ボタンの動作を選択するオプションを追加。「アプリを終了する」（デフォルト）または「タスクトレイに最小化する」を選べます。最小化を選択した場合、タスクトレイから終了する方法を案内する注意書きが表示されます。
