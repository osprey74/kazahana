# Release Notes — kazahana v2.1.3

## New Features

- **Notification image thumbnails**: Posts with images now show small square-cropped thumbnails (32x32px) on the right side of each notification item, supporting up to 4 images side by side.

## Improvements

- **BSAF post spacing**: BSAF-compatible bot posts now have additional vertical margin (10px top/bottom) for better visual separation in the timeline.

## Bug Fixes

- **BSAF demo**: Renamed "生テキスト" to "RAWテキスト" for clarity in the BSAF demo page.

# リリースノート — kazahana v2.1.3

## 新機能

- **通知欄の画像サムネイル**: 画像付き投稿の通知に、32x32px の正方形にトリミングされたサムネイルを通知行の右側に表示するようになりました。最大4枚まで横並びで表示されます。

## 改善

- **BSAFポストの余白**: BSAF対応Botの投稿に上下マージン（各10px）を追加し、タイムライン上での視認性を向上しました。

## バグ修正

- **BSAFデモ**: BSAFデモページ内の「生テキスト」を「RAWテキスト」に修正しました。
