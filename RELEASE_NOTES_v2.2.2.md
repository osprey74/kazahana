# Release Notes — v2.2.2

## ✨ New Features

- **ALT text display on post cards**: Images with ALT text now show the first 128 characters on the post card. Full ALT text is viewable in the image lightbox.
- **Language sync from Bluesky**: App language is now detected from the user's recent post language tags. Falls back to device locale if no match is found or if the user has manually set a language in kazahana settings.
- **Post drafts (up to 20)**: Save post drafts including text and image thumbnails. Drafts can be loaded, individually deleted, or cleared all at once from the compose dialog.

## 📝 Documentation

- Updated internal design spec (`kazahana-spec.md`) and added mobile spec
- Updated README (EN/JA) with new features

# リリースノート — v2.2.2

## ✨ 新機能

- **投稿カードへの ALT テキスト表示**: ALT テキスト付き画像の先頭128文字を投稿カード上に表示。全文はライトボックスで確認可能。
- **Bluesky 言語設定からのアプリ言語同期**: ユーザーの最近の投稿の言語タグからアプリ言語を自動検出。マッチしない場合や手動設定済みの場合は端末ロケールにフォールバック。
- **投稿の下書き保存（最大20件）**: テキストと画像サムネイルを含む下書きを保存。投稿フォームから読み込み・個別削除・一括削除が可能。
