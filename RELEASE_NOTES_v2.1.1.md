# Release Notes — v2.1.1

## English

### ✨ New Features

- **"Search Web" in context menu** — When selecting text in the post detail view, a new "Search Web" option appears in the right-click menu. It opens a Google search for the selected text in your default browser. Supported in all 11 languages.
- **Clickable URLs in DM messages** — URLs in direct messages are now rendered as clickable links that open in your default browser.
- **Clickable hashtags in DM messages** — Hashtags in direct messages are now clickable, navigating to search results for that hashtag (same behavior as timeline posts).
- **Clickable mentions in DM messages** — Mentions in direct messages are now clickable, navigating to the mentioned user's profile.

### 🐛 Bug Fixes

- **BSAF demo: hide tags when parsing is disabled** — BSAF tags and divider are no longer shown in the demo when parsing is turned off.

### 📝 Documentation

- Added BSAF demo page link to Japanese README, Japanese user guide, and Japanese landing page.
- Renamed demo settings section label to "RAWデータ".

## 日本語

### ✨ 新機能

- **コンテキストメニューに「Webで検索」を追加** — ポスト詳細画面でテキストを選択して右クリックすると、「Webで検索」メニューが表示されます。選択テキストをデフォルトブラウザでGoogle検索します。全11言語対応。
- **DMメッセージ内のURLをクリッカブル化** — ダイレクトメッセージ内のURLがリンクとして表示され、クリックでデフォルトブラウザで開きます。
- **DMメッセージ内のハッシュタグをクリッカブル化** — ダイレクトメッセージ内のハッシュタグがクリック可能になり、タイムラインのポストと同様にそのハッシュタグの検索結果に遷移します。
- **DMメッセージ内のメンションをクリッカブル化** — ダイレクトメッセージ内のメンションがクリック可能になり、該当ユーザーのプロフィール画面に遷移します。

### 🐛 バグ修正

- **BSAFデモ: パース無効時にタグを非表示** — デモページでパースをオフにした際、BSAFタグと区切り線が表示されなくなりました。

### 📝 ドキュメント

- 日本語README、日本語操作マニュアル、日本語ランディングページにBSAFデモページへのリンクを追加しました。
- デモ設定セクションのラベルを「RAWデータ」に変更しました。
