kazahana

가볍고, 빠르게.
Bluesky를 더 쾌적하게.

kazahana는 가벼운 Bluesky 데스크톱 클라이언트입니다.
OS 네이티브 기술로 메모리 사용을 최소화합니다.
당신의 Bluesky 경험을 더 심플하게.

desktop_windows Windows용 다운로드 laptop_mac macOS (준비 중)

무료 & 오픈 소스 (MIT 라이선스)

특징

kazahana가 매일의 Bluesky 경험을 더 좋게 만드는 네 가지 이유

memory

가벼움

OS 기본 WebView를 활용하여 PWA 대비 메모리 사용량을 대폭 절감합니다. 오랜 시간 사용해도 PC가 느려지지 않습니다.

bolt

빠름

네이티브 앱의 시작 속도. 타임라인 로딩이 매끄럽고 대기 시간 없이 피드를 탐색할 수 있습니다.

dashboard

심플

깔끔한 1컬럼 UI. 타임라인, 게시, 알림, 검색, 프로필 — 필요한 기능은 모두 갖추고 있습니다.

translate

다국어 지원

한국어와 일본어를 포함한 11개 언어를 지원합니다. 전 세계 Bluesky 사용자가 모국어로 kazahana를 사용할 수 있습니다.

기능 소개

kazahana에서 사용할 수 있는 모든 기능

dynamic_feed 타임라인 자동 새로고침 (간격 설정 가능)
bookmark 읽기 위치 마커
edit_square 게시 및 답글
favorite 좋아요 및 리포스트
image 이미지 첨부 (최대 4장, ALT 텍스트 지원)
zoom_in 이미지 라이트박스
link 링크 카드 (OGP 미리보기)
forum 스레드 보기
notifications 알림 (좋아요, 리포스트, 팔로우, 멘션, 답글, 인용)
person 프로필 및 팔로우 / 언팔로우
thumb_up 좋아요한 게시물
photo_library 미디어 게시물
search 게시물 및 사용자 검색
dark_mode 다크 / 라이트 / 시스템 테마
translate 11개 언어 지원
shield 콘텐츠 관리 (라벨, 블러, 설정)
tab 맞춤 피드 및 리스트 피드
lock 스레드게이트 및 포스트게이트 (답글/인용 제한)
delete 게시물 삭제
refresh 수동 새로고침 (탭 클릭 / F5 / 버튼)
vertical_align_top 시스템 트레이
