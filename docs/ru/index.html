kazahana — лёгкий десктопный клиент для Bluesky.
Построен на основе нативных технологий ОС для минимального потребления памяти.
Ваш опыт в Bluesky, упрощённый.

Возможности

Четыре причины, по которым kazahana делает ваш ежедневный опыт в Bluesky лучше

memory

Лёгкий

Использует нативный WebView вашей ОС, значительно снижая потребление памяти по сравнению с PWA. Ваш ПК остаётся быстрым даже после многих часов работы.

bolt

Быстрый

Скорость запуска нативного приложения. Плавная загрузка ленты без ожидания — листайте посты без задержек.

dashboard

Простой

Чистый одноколоночный интерфейс. Лента, публикации, уведомления, поиск и профиль — всё необходимое, ничего лишнего.

translate

Многоязычный

Поддержка 11 языков, включая русский и японский. Пользователи Bluesky по всему миру могут использовать kazahana на своём родном языке.

Возможности

Всё, что доступно в kazahana

dynamic_feed Автоматическое обновление ленты (настраиваемый интервал)
bookmark Маркер позиции чтения
edit_square Публикация и ответы
favorite Лайки и репосты
image Вложение изображений (до 4, с ALT-текстом)
zoom_in Просмотр изображений
link Карточка ссылки (OGP-предпросмотр)
forum Просмотр тредов
notifications Уведомления (лайк, репост, подписка, упоминание, ответ, цитата)
person Профиль и подписка / отписка
thumb_up Понравившиеся публикации
photo_library Медиа публикации
search Поиск постов и пользователей
dark_mode Тёмная / светлая / системная тема
translate 11 языков
shield Модерация контента (метки, размытие, настройки)
tab Пользовательские ленты и ленты списков
lock Threadgate и postgate (ограничения ответов/цитат)
delete Удаление публикаций
refresh Ручное обновление (клик по вкладке / F5 / кнопка)
vertical_align_top Системный трей
