Руководство по установке

kazahana в настоящее время не имеет цифровой подписи, поэтому ваша ОС может отображать предупреждения безопасности.
Следуйте инструкциям ниже для установки.

desktop_windows Windows

1. Скачайте установщик

Скачайте последний установщик со страницы GitHub Releases.

Имя файлаФорматЦелевая платформа
kazahana_x.x.x_x64-setup.exeУстановщик NSISWindows 10 / 11 (64-бит)
kazahana_x.x.x_x64_en-US.msiУстановщик MSIWindows 10 / 11 (64-бит)

2. Обработка предупреждения SmartScreen

При запуске установщика Microsoft Defender SmartScreen может отобразить предупреждение:

Система Windows защитила ваш компьютер
Фильтр Microsoft Defender SmartScreen предотвратил запуск неопознанного приложения. Запуск этого приложения может подвергнуть ваш компьютер риску.

Это стандартное предупреждение для приложений без цифровой подписи. kazahana — это программное обеспечение с открытым исходным кодом, доступное на GitHub.

  1. Нажмите “Подробнее” в диалоговом окне предупреждения
  2. Отобразится имя приложения и информация об издателе
  3. Нажмите кнопку “Выполнить в любом случае”

3. Установка

После прохождения SmartScreen запустится стандартный мастер установки.

  1. Подтвердите папку установки (значение по умолчанию подходит)
  2. Нажмите “Установить”
  3. После завершения установки нажмите “Готово”

4. Запуск

Запустите kazahana из меню «Пуск» или с помощью ярлыка на рабочем столе. Предупреждение SmartScreen не будет появляться после первого запуска.

Примечание: Предупреждения браузера при скачивании

  • Microsoft Edge: Может показать “Этот файл скачивается нечасто.” Нажмите на меню … и выберите “Сохранить”.
  • Google Chrome: Может показать “Этот файл может быть опасным.” Нажмите на значок ∧ и выберите “Сохранить”.

laptop_mac macOS

1. Скачайте установщик

Скачайте последний установщик со страницы GitHub Releases.

Имя файлаЦелевая платформа
kazahana_x.x.x_x64.dmgmacOS (Intel)
kazahana_x.x.x_aarch64.dmgmacOS (Apple Silicon: M1 / M2 / M3 / M4)

Чтобы узнать чип вашего Mac: Нажмите на меню Apple в левом верхнем углу и выберите “Об этом Mac”.

2. Установка приложения

  1. Откройте скачанный файл .dmg
  2. Перетащите значок kazahana в папку Программы
  3. Закройте окно .dmg и извлеките образ диска

3. Обработка предупреждения Gatekeeper

При первом запуске macOS Gatekeeper отобразит предупреждение:

Не удаётся открыть “kazahana”, так как не удалось подтвердить разработчика.

Это стандартное предупреждение для приложений без цифровой подписи или нотаризации.

Способ A: Разрешить через Системные настройки (Рекомендуется)

  1. Когда появится предупреждение, нажмите “OK”, чтобы закрыть диалог
  2. Откройте Системные настройки > “Конфиденциальность и безопасность”
  3. Внизу вы увидите: “Использование kazahana заблокировано, так как приложение не от установленного разработчика”
  4. Нажмите “Всё равно открыть”
  5. Подтвердите паролем администратора или Touch ID
  6. Когда появится диалог подтверждения, нажмите “Открыть”

Способ B: Открыть через контекстное меню

  1. Откройте папку Программы в Finder
  2. Щёлкните правой кнопкой мыши (или Control + щелчок) по kazahana
  3. Выберите “Открыть” в контекстном меню
  4. При появлении запроса нажмите “Открыть”

В некоторых версиях macOS Способ B может не работать. В этом случае попробуйте Способ A.

Способ C: Удаление атрибута карантина через Терминал (Для опытных пользователей)

Откройте Терминал и выполните:

xattr -cr /Applications/kazahana.app

Эта команда удаляет атрибут карантина (com.apple.quarantine), применяемый к загруженным файлам, что позволяет обойти проверку Gatekeeper.

4. Запуск

Запустите kazahana из Launchpad или папки Программы. После первоначального разрешения приложение будет запускаться нормально.

delete Удаление

Windows

Перейдите в Параметры > Приложения > Установленные приложения, найдите kazahana и нажмите “Удалить”.

macOS

Перетащите kazahana из папки Программы в Корзину.

build Устранение неполадок

Приложение не запускается

  • Убедитесь, что ваша ОС обновлена до последней версии
  • Windows: Требуется Windows 10 или новее
  • macOS: Требуется macOS 12 (Monterey) или новее

“kazahana повреждён и не может быть открыт” (macOS)

Выполните следующую команду в Терминале:

xattr -cr /Applications/kazahana.app

shield О безопасности

kazahana — это программное обеспечение с открытым исходным кодом (лицензия MIT). Исходный код общедоступен на GitHub и может быть проверен любым желающим.

Приложение в настоящее время не имеет цифровой подписи, поэтому ваша ОС отображает предупреждения безопасности. Это не означает, что приложение опасно — это лишь означает, что издатель не может быть подтверждён электронным способом. Цифровая подпись запланирована на будущее.
