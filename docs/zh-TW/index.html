kazahana

輕巧、快速。
更好的 Bluesky 體驗。

kazahana 是一款輕量級 Bluesky 桌面用戶端。
採用作業系統原生技術，將記憶體使用降至最低。
您的 Bluesky 體驗，從此更加簡單。

desktop_windows 下載 Windows 版 laptop_mac 下載 macOS 版

免費且開源（MIT 授權）

特色功能

四大特色，讓您的每日 Bluesky 體驗更上一層樓

memory

輕量

使用作業系統原生 WebView，與 PWA 相比大幅降低記憶體使用量。即使長時間使用，電腦依然流暢。

bolt

快速

原生應用程式的啟動速度。時間軸載入順暢無延遲 — 輕鬆瀏覽您的動態。

dashboard

簡潔

清爽的單欄介面。時間軸、發文、通知、搜尋、個人檔案 — 該有的功能一應俱全。

translate

多語言

支援包含繁體中文與日文在內的 11 種語言。全球 Bluesky 使用者都能以母語使用 kazahana。

功能一覽

kazahana 提供的所有功能

dynamic_feed 時間軸自動更新（可設定間隔）
bookmark 閱讀位置標記
edit_square 發佈與回覆
favorite 按讚與轉發
image 圖片附件（最多 4 張，支援 ALT 文字）
zoom_in 圖片燈箱
link 連結卡片（OGP 預覽）
forum 串文檢視
notifications 通知（按讚、轉發、追蹤、提及、回覆、引用）
person 個人檔案與追蹤 / 取消追蹤
thumb_up 喜歡的貼文
photo_library 媒體貼文
search 搜尋貼文與使用者
dark_mode 深色 / 淺色 / 跟隨系統主題
translate 支援 11 種語言
shield 內容審核（標籤判定、模糊處理、設定）
tab 自訂動態消息與列表動態消息
lock 串文門與發佈門（回覆/引用限制）
delete 刪除貼文
refresh 手動重新整理（點擊分頁 / F5 / 按鈕）
vertical_align_top 系統匣常駐
