安装指南

kazahana 目前尚未进行代码签名，因此您的操作系统可能会显示安全警告。
请按照以下步骤进行安装。

desktop_windows Windows

1. 下载安装程序

从 GitHub Releases 页面下载最新的安装程序。

文件名格式适用平台
kazahana_x.x.x_x64-setup.exeNSIS 安装程序Windows 10 / 11（64 位）
kazahana_x.x.x_x64_en-US.msiMSI 安装程序Windows 10 / 11（64 位）

2. 处理 SmartScreen 警告

运行安装程序时，Microsoft Defender SmartScreen 可能会显示警告：

Windows 已保护你的电脑
Microsoft Defender SmartScreen 已阻止无法识别的应用启动。运行此应用可能会使你的电脑面临风险。

这是未进行代码签名的应用的标准警告。kazahana 是开源软件，其源代码已公开在 GitHub 上。

  1. 在警告对话框中点击"更多信息"
  2. 将会显示应用名称和发布者信息
  3. 点击"仍要运行"按钮

3. 安装

通过 SmartScreen 后，标准安装向导将会启动。

  1. 确认安装文件夹（使用默认值即可）
  2. 点击"安装"
  3. 安装完成后，点击"完成"

4. 启动

从开始菜单或桌面快捷方式启动 kazahana。首次启动后，SmartScreen 警告将不再出现。

注意：浏览器下载警告

  • Microsoft Edge：可能会显示"此文件不常被下载。"点击 … 菜单并选择"保留"。
  • Google Chrome：可能会显示"此文件可能有危险。"点击 ∧ 图标并选择"保留"。

laptop_mac macOS

1. 下载安装程序

从 GitHub Releases 页面下载最新的安装程序。

文件名适用平台
kazahana_x.x.x_x64.dmgmacOS（Intel）
kazahana_x.x.x_aarch64.dmgmacOS（Apple Silicon：M1 / M2 / M3 / M4）

如何查看 Mac 的芯片：点击左上角的苹果菜单，选择"关于本机"。

2. 安装应用

  1. 打开下载的 .dmg 文件
  2. 将 kazahana 图标拖拽至应用程序文件夹
  3. 关闭 .dmg 窗口并推出磁盘映像

3. 处理 Gatekeeper 警告

首次启动时，macOS Gatekeeper 会显示警告：

无法打开"kazahana"，因为无法验证开发者。

这是未进行代码签名或公证的应用的标准警告。

方法 A：通过系统设置允许（推荐）

  1. 当警告出现时，点击"好"关闭对话框
  2. 打开系统设置 > "隐私与安全性"
  3. 在底部，您会看到："kazahana 因为不是来自已识别的开发者而被阻止"
  4. 点击"仍要打开"
  5. 使用管理员密码或 Touch ID 进行验证
  6. 当确认对话框出现时，点击"打开"

方法 B：通过右键点击打开

  1. 在 Finder 中打开应用程序文件夹
  2. 在 kazahana 上右键点击（或 Control + 点击）
  3. 从快捷菜单中选择"打开"
  4. 当提示出现时，点击"打开"

在某些 macOS 版本中，方法 B 可能无法使用。如果是这样，请尝试方法 A。

方法 C：通过终端移除隔离属性（高级）

打开终端并运行：

xattr -cr /Applications/kazahana.app

这会移除应用于下载文件的隔离属性（com.apple.quarantine），让您可以绕过 Gatekeeper 验证。

4. 启动

从 Launchpad 或应用程序文件夹启动 kazahana。完成初始权限步骤后，它将正常启动。

delete 卸载

Windows

前往设置 > 应用 > 已安装的应用，找到 kazahana，然后点击"卸载"。

macOS

将 kazahana 从应用程序文件夹拖拽至废纸篓。

build 故障排除

应用无法启动

  • 请确认您的操作系统已更新至最新版本
  • Windows：需要 Windows 10 或更高版本
  • macOS：需要 macOS 12（Monterey）或更高版本

"kazahana 已损坏，无法打开"（macOS）

在终端中运行以下命令：

xattr -cr /Applications/kazahana.app
