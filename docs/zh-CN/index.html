kazahana

轻巧、快速。
更好的 Bluesky 体验。

kazahana 是一款轻量级 Bluesky 桌面客户端。
采用操作系统原生技术，将内存使用降至最低。
您的 Bluesky 体验，从此更加简单。

免费且开源（MIT 许可）

特色功能

四大特色，让您的每日 Bluesky 体验更上一层楼

memory

轻量

使用操作系统原生 WebView，与 PWA 相比大幅降低内存使用量。即使长时间使用，电脑依然流畅。

bolt

快速

原生应用程序的启动速度。时间线加载顺畅无延迟——轻松浏览您的动态。

dashboard

简洁

清爽的单栏界面。时间线、发帖、通知、搜索、个人资料——该有的功能一应俱全。

translate

多语言

支持包含简体中文与日文在内的 11 种语言。全球 Bluesky 用户都能以母语使用 kazahana。

功能一览

kazahana 提供的所有功能

dynamic_feed 时间线自动更新（可配置间隔）
bookmark 阅读位置标记
edit_square 发布与回复
favorite 点赞与转发
image 图片附件（最多 4 张，支持 ALT 文字）
zoom_in 图片灯箱
link 链接卡片（OGP 预览）
forum 帖子串查看
notifications 通知（点赞、转发、关注、提及、回复、引用）
person 个人资料与关注 / 取消关注
thumb_up 喜欢的帖子
photo_library 媒体帖子
search 搜索帖子和用户
dark_mode 深色 / 浅色 / 跟随系统主题
translate 支持 11 种语言
shield 内容审核（标签判定、模糊处理、设置）
tab 自定义信息流与列表信息流
lock 串门与发布门（回复/引用限制）
delete 删除帖子
refresh 手动刷新（点击标签页 / F5 / 按钮）
vertical_align_top 系统托盘常驻
