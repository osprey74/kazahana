Guia de Instalação

O kazahana não possui assinatura de código atualmente, então seu sistema operacional pode exibir avisos de segurança.
Siga os passos abaixo para instalar.

desktop_windows Windows

1. Baixar o Instalador

Baixe o instalador mais recente na página de Releases do GitHub.

Nome do ArquivoFormatoDestino
kazahana_x.x.x_x64-setup.exeInstalador NSISWindows 10 / 11 (64-bit)
kazahana_x.x.x_x64_en-US.msiInstalador MSIWindows 10 / 11 (64-bit)

2. Lidando com o Aviso do SmartScreen

Ao executar o instalador, o Microsoft Defender SmartScreen pode exibir um aviso:

O Windows protegeu seu computador
O Microsoft Defender SmartScreen impediu a execução de um aplicativo não reconhecido. Executar este aplicativo pode colocar seu computador em risco.

Este é um aviso padrão para aplicativos que não possuem assinatura de código. O kazahana é um software de código aberto com seu código-fonte disponível publicamente no GitHub.

  1. Clique em “Mais informações” na caixa de diálogo de aviso
  2. O nome do aplicativo e as informações do editor serão exibidos
  3. Clique no botão “Executar mesmo assim”

3. Instalação

Após passar pelo SmartScreen, o assistente de instalação padrão será iniciado.

  1. Confirme a pasta de instalação (o padrão está correto)
  2. Clique em “Instalar”
  3. Quando a instalação for concluída, clique em “Concluir”

4. Iniciar

Inicie o kazahana pelo Menu Iniciar ou pelo atalho na área de trabalho. O aviso do SmartScreen não aparecerá após a primeira execução.

Nota: Avisos de Download do Navegador

  • Microsoft Edge: Pode exibir “Este arquivo não é baixado com frequência.” Clique no menu … e selecione “Manter”.
  • Google Chrome: Pode exibir “Este arquivo pode ser perigoso.” Clique no ícone ∧ e selecione “Manter”.

laptop_mac macOS

1. Baixar o Instalador

Baixe o instalador mais recente na página de Releases do GitHub.

Nome do ArquivoDestino
kazahana_x.x.x_x64.dmgmacOS (Intel)
kazahana_x.x.x_aarch64.dmgmacOS (Apple Silicon: M1 / M2 / M3 / M4)

Para verificar o chip do seu Mac: Clique no menu Apple no canto superior esquerdo e selecione “Sobre Este Mac”.

2. Instalar o Aplicativo

  1. Abra o arquivo .dmg baixado
  2. Arraste o ícone do kazahana para a pasta Aplicativos
  3. Feche a janela do .dmg e ejete a imagem de disco

3. Lidando com o Aviso do Gatekeeper

Na primeira execução, o macOS Gatekeeper exibirá um aviso:

“kazahana” não pode ser aberto porque o desenvolvedor não pode ser verificado.

Este é um aviso padrão para aplicativos que não possuem assinatura de código ou notarização.

Método A: Permitir via Configurações do Sistema (Recomendado)

  1. Quando o aviso aparecer, clique em “OK” para fechar a caixa de diálogo
  2. Abra Configurações do Sistema > “Privacidade e Segurança”
  3. Na parte inferior, você verá: “kazahana foi bloqueado porque não é de um desenvolvedor identificado”
  4. Clique em “Abrir Mesmo Assim”
  5. Autentique-se com sua senha de administrador ou Touch ID
  6. Quando a caixa de diálogo de confirmação aparecer, clique em “Abrir”

Método B: Abrir com Clique Direito

  1. Abra a pasta Aplicativos no Finder
  2. Clique com o botão direito (ou Control + clique) no kazahana
  3. Selecione “Abrir” no menu de contexto
  4. Quando solicitado, clique em “Abrir”

Em algumas versões do macOS, o Método B pode não funcionar. Se isso acontecer, tente o Método A.

Método C: Remover Atributo de Quarentena via Terminal (Avançado)

Abra o Terminal e execute:

xattr -cr /Applications/kazahana.app

Isso remove o atributo de quarentena (com.apple.quarantine) aplicado a arquivos baixados, permitindo ignorar a verificação do Gatekeeper.

4. Iniciar

Inicie o kazahana pelo Launchpad ou pela pasta Aplicativos. Após a etapa inicial de permissão, ele será iniciado normalmente.

delete Desinstalar

Windows

Vá para Configurações > Aplicativos > Aplicativos instalados, encontre o kazahana e clique em “Desinstalar”.

macOS

Arraste o kazahana da pasta Aplicativos para a Lixeira.

build Solução de Problemas

O aplicativo não inicia

  • Certifique-se de que seu sistema operacional está atualizado para a versão mais recente
  • Windows: É necessário Windows 10 ou posterior
  • macOS: É necessário macOS 12 (Monterey) ou posterior

“kazahana está danificado e não pode ser aberto” (macOS)

Execute o seguinte comando no Terminal:

xattr -cr /Applications/kazahana.app

shield Sobre Segurança

O kazahana é um software de código aberto (Licença MIT). O código-fonte está disponível publicamente no GitHub e pode ser revisado por qualquer pessoa.

O aplicativo não possui assinatura de código atualmente, por isso seu sistema operacional exibe avisos de segurança. Isso não significa que o aplicativo é perigoso — significa apenas que o editor não pode ser verificado eletronicamente. A assinatura de código está planejada para o futuro.
