## Legend

| Symbol | Meaning |
|--------|---------|
| ✅ | 実装済み |
| 🚧 | 対応中 / 部分実装 |
| ⬜ | 未実装（対応予定） |
| N/A | プラットフォーム非対象（仕様上不要） |
| ❓ | 要確認（ソースから判定不能） |

> **Windows / macOS** は同一バイナリのため基本的に同一扱いです。  
> macOS 固有の挙動のみ `macOS のみ` と注記します。

## 更新ルール

1. 新機能追加・バグ修正 PR のマージ時は **このファイルを必ず更新** する
2. ステータスは `✅ → ⬜ → 🚧 → ✅` の順で遷移させる
3. `❓` は次回の確認時に `✅` か `⬜` に解消する（放置しない）
4. `N/A` の再分類は プラットフォーム仕様変更時のみ行う
