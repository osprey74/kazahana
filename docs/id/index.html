kazahana

Ringan. Cepat.
Cara lebih baik menikmati Bluesky.

kazahana adalah klien desktop ringan untuk Bluesky.
Dibangun dengan teknologi native OS untuk penggunaan memori minimal.
Pengalaman Bluesky Anda, disederhanakan.

desktop_windows Unduh untuk Windows laptop_mac Unduh untuk macOS

Gratis & open source (Lisensi MIT)

Fitur

Empat alasan kazahana membuat pengalaman Bluesky harian Anda lebih baik

memory

Ringan

Menggunakan WebView native OS Anda untuk mengurangi penggunaan memori secara drastis dibandingkan PWA. PC Anda tetap cepat, bahkan setelah berjam-jam digunakan.

bolt

Cepat

Kecepatan startup aplikasi native. Timeline dimuat dengan lancar tanpa menunggu — jelajahi feed Anda tanpa hambatan.

dashboard

Sederhana

UI satu kolom yang bersih. Timeline, posting, notifikasi, pencarian, dan profil — semua yang Anda butuhkan, tanpa yang berlebihan.

translate

Multibahasa

Mendukung 11 bahasa termasuk bahasa Indonesia dan Jepang. Pengguna Bluesky di seluruh dunia dapat menggunakan kazahana dalam bahasa ibu mereka.
