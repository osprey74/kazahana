Panduan Instalasi

kazahana saat ini belum memiliki tanda tangan kode (code-signed), sehingga sistem operasi Anda mungkin menampilkan peringatan keamanan.
Ikuti langkah-langkah di bawah ini untuk menginstal.

desktop_windows Windows

1. Unduh Installer

Unduh installer terbaru dari halaman GitHub Releases.

Nama FileFormatTarget
kazahana_x.x.x_x64-setup.exeInstaller NSISWindows 10 / 11 (64-bit)
kazahana_x.x.x_x64_en-US.msiInstaller MSIWindows 10 / 11 (64-bit)

2. Menangani Peringatan SmartScreen

Saat Anda menjalankan installer, Microsoft Defender SmartScreen mungkin menampilkan peringatan:

Windows melindungi PC Anda
Microsoft Defender SmartScreen mencegah aplikasi yang tidak dikenal untuk berjalan. Menjalankan aplikasi ini mungkin membahayakan PC Anda.

Ini adalah peringatan standar untuk aplikasi yang belum memiliki tanda tangan kode. kazahana adalah perangkat lunak sumber terbuka dengan kode sumber yang tersedia secara publik di GitHub.

  1. Klik “Info selengkapnya” pada dialog peringatan
  2. Nama aplikasi dan informasi penerbit akan ditampilkan
  3. Klik tombol “Tetap jalankan”

3. Instalasi

Setelah melewati SmartScreen, wizard instalasi standar akan dimulai.

  1. Konfirmasi folder instalasi (default sudah cukup)
  2. Klik “Install”
  3. Setelah instalasi selesai, klik “Finish”

4. Jalankan

Jalankan kazahana dari Start Menu atau pintasan desktop. Peringatan SmartScreen tidak akan muncul lagi setelah peluncuran pertama.

Catatan: Peringatan Unduhan Browser

  • Microsoft Edge: Mungkin menampilkan “File ini tidak umum diunduh.” Klik menu … dan pilih “Simpan”.
  • Google Chrome: Mungkin menampilkan “File ini mungkin berbahaya.” Klik ikon ∧ dan pilih “Simpan”.

laptop_mac macOS

1. Unduh Installer

Unduh installer terbaru dari halaman GitHub Releases.

Nama FileTarget
kazahana_x.x.x_x64.dmgmacOS (Intel)
kazahana_x.x.x_aarch64.dmgmacOS (Apple Silicon: M1 / M2 / M3 / M4)

Untuk memeriksa chip Mac Anda: Klik menu Apple di pojok kiri atas dan pilih “Mengenai Mac Ini”.

2. Instal Aplikasi

  1. Buka file .dmg yang telah diunduh
  2. Seret ikon kazahana ke folder Applications
  3. Tutup jendela .dmg dan keluarkan disk image

3. Menangani Peringatan Gatekeeper

Pada peluncuran pertama, macOS Gatekeeper akan menampilkan peringatan:

“kazahana” tidak dapat dibuka karena pengembang tidak dapat diverifikasi.

Ini adalah peringatan standar untuk aplikasi yang belum memiliki tanda tangan kode atau notarisasi.

Metode A: Izinkan melalui Pengaturan Sistem (Disarankan)

  1. Saat peringatan muncul, klik “OK” untuk menutup dialog
  2. Buka Pengaturan Sistem > “Privasi & Keamanan”
  3. Di bagian bawah, Anda akan melihat: “kazahana diblokir karena bukan dari pengembang yang teridentifikasi”
  4. Klik “Buka Tetap”
  5. Autentikasi dengan kata sandi admin atau Touch ID
  6. Saat dialog konfirmasi muncul, klik “Buka”

Metode B: Buka melalui Klik Kanan

  1. Buka folder Applications di Finder
  2. Klik kanan (atau Control + klik) pada kazahana
  3. Pilih “Buka” dari menu konteks
  4. Saat diminta, klik “Buka”

Pada beberapa versi macOS, Metode B mungkin tidak berfungsi. Jika demikian, coba Metode A.

Metode C: Hapus Atribut Karantina melalui Terminal (Lanjutan)

Buka Terminal dan jalankan:

xattr -cr /Applications/kazahana.app

Perintah ini menghapus atribut karantina (com.apple.quarantine) yang diterapkan pada file yang diunduh, sehingga Anda dapat melewati verifikasi Gatekeeper.

4. Jalankan

Jalankan kazahana dari Launchpad atau folder Applications. Setelah langkah perizinan awal, aplikasi akan berjalan secara normal.

shield Tentang Keamanan

kazahana adalah perangkat lunak sumber terbuka (Lisensi MIT). Kode sumber tersedia secara publik di GitHub dan dapat ditinjau oleh siapa saja.

Aplikasi ini saat ini belum memiliki tanda tangan kode, itulah sebabnya sistem operasi Anda menampilkan peringatan keamanan. Ini bukan berarti aplikasi ini berbahaya — ini hanya berarti penerbit tidak dapat diverifikasi secara elektronik. Tanda tangan kode direncanakan untuk masa mendatang.
