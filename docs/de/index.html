kazahana ist ein leichter Bluesky Desktop-Client.
Gebaut auf nativer OS-Technologie für minimalen Speicherverbrauch.
Dein Bluesky-Erlebnis, vereinfacht.

Funktionen

Vier Gründe, warum kazahana dein tägliches Bluesky-Erlebnis verbessert

memory

Leicht

Nutzt den nativen WebView deines Betriebssystems und reduziert den Speicherverbrauch im Vergleich zu PWA drastisch. Dein PC bleibt schnell, auch nach stundenlanger Nutzung.

bolt

Schnell

Native App-Startgeschwindigkeit. Flüssiges Timeline-Laden ohne Wartezeit — stöbere durch deinen Feed ohne Verzögerung.

dashboard

Einfach

Eine aufgeräumte einspaltiges UI. Timeline, Beiträge, Benachrichtigungen, Suche und Profil — alles was du brauchst, nichts Überflüssiges.

translate

Mehrsprachig

Unterstützt 11 Sprachen, darunter Deutsch und Japanisch. Bluesky-Nutzer weltweit können kazahana in ihrer Muttersprache verwenden.
