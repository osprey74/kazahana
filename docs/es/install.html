Guía de instalación

kazahana no cuenta actualmente con firma de código, por lo que tu sistema operativo podría mostrar advertencias de seguridad.
Sigue los pasos a continuación para instalarlo.

desktop_windows Windows

1. Descargar el instalador

Descarga el instalador más reciente desde la página de GitHub Releases.

Nombre del archivoFormatoObjetivo
kazahana_x.x.x_x64-setup.exeInstalador NSISWindows 10 / 11 (64 bits)
kazahana_x.x.x_x64_en-US.msiInstalador MSIWindows 10 / 11 (64 bits)

2. Manejo de la advertencia de SmartScreen

Al ejecutar el instalador, Microsoft Defender SmartScreen podría mostrar una advertencia:

Windows protegió tu PC
Microsoft Defender SmartScreen impidió que se iniciara una aplicación no reconocida. Ejecutar esta aplicación podría poner en riesgo tu PC.

Esta es una advertencia estándar para aplicaciones sin firma de código. kazahana es software de código abierto con su código fuente disponible públicamente en GitHub.

  1. Haz clic en “Más información” en el diálogo de advertencia
  2. Se mostrará el nombre de la aplicación y la información del editor
  3. Haz clic en el botón “Ejecutar de todas formas”

3. Instalación

Después de pasar SmartScreen, se iniciará el asistente de instalación estándar.

  1. Confirma la carpeta de instalación (la predeterminada está bien)
  2. Haz clic en “Instalar”
  3. Una vez completada la instalación, haz clic en “Finalizar”

4. Ejecución

Ejecuta kazahana desde el menú Inicio o el acceso directo del escritorio. La advertencia de SmartScreen no aparecerá después de la primera ejecución.

Nota: Advertencias de descarga del navegador

  • Microsoft Edge: Puede mostrar “Este archivo no se descarga comúnmente.” Haz clic en el menú … y selecciona “Conservar”.
  • Google Chrome: Puede mostrar “Este archivo puede ser peligroso.” Haz clic en el icono ∧ y selecciona “Conservar”.

laptop_mac macOS

1. Descargar el instalador

Descarga el instalador más reciente desde la página de GitHub Releases.

Nombre del archivoObjetivo
kazahana_x.x.x_x64.dmgmacOS (Intel)
kazahana_x.x.x_aarch64.dmgmacOS (Apple Silicon: M1 / M2 / M3 / M4)

Para verificar el chip de tu Mac: Haz clic en el menú Apple en la esquina superior izquierda y selecciona “Acerca de este Mac”.

2. Instalar la aplicación

  1. Abre el archivo .dmg descargado
  2. Arrastra el icono de kazahana a la carpeta de Aplicaciones
  3. Cierra la ventana del .dmg y expulsa la imagen de disco

3. Manejo de la advertencia de Gatekeeper

En la primera ejecución, macOS Gatekeeper mostrará una advertencia:

No se puede abrir “kazahana” porque no se puede verificar al desarrollador.

Esta es una advertencia estándar para aplicaciones sin firma de código ni notarización.

Método A: Permitir desde Ajustes del Sistema (Recomendado)

  1. Cuando aparezca la advertencia, haz clic en “OK” para cerrar el diálogo
  2. Abre Ajustes del Sistema > “Privacidad y seguridad”
  3. En la parte inferior, verás: “Se ha bloqueado el uso de kazahana porque no es de un desarrollador identificado”
  4. Haz clic en “Abrir de todas formas”
  5. Autentícate con tu contraseña de administrador o Touch ID
  6. Cuando aparezca el diálogo de confirmación, haz clic en “Abrir”

Método B: Abrir con clic derecho

  1. Abre la carpeta de Aplicaciones en Finder
  2. Haz clic derecho (o Control + clic) en kazahana
  3. Selecciona “Abrir” en el menú contextual
  4. Cuando se te solicite, haz clic en “Abrir”

En algunas versiones de macOS, el Método B podría no funcionar. Si es así, prueba el Método A.

Método C: Eliminar el atributo de cuarentena vía Terminal (Avanzado)

Abre Terminal y ejecuta:

xattr -cr /Applications/kazahana.app

Esto elimina el atributo de cuarentena (com.apple.quarantine) aplicado a los archivos descargados, permitiéndote eludir la verificación de Gatekeeper.

4. Ejecución

Ejecuta kazahana desde Launchpad o la carpeta de Aplicaciones. Después del paso de permisos inicial, se ejecutará normalmente.

delete Desinstalar

Windows

macOS

build Solución de problemas

La aplicación no se ejecuta

  • Asegúrate de que tu sistema operativo esté actualizado a la última versión
  • Windows: Se requiere Windows 10 o posterior
  • macOS: Se requiere macOS 12 (Monterey) o posterior

“kazahana está dañado y no se puede abrir” (macOS)

Ejecuta el siguiente comando en Terminal:

xattr -cr /Applications/kazahana.app

shield Sobre la seguridad

kazahana es software de código abierto (Licencia MIT). El código fuente está disponible públicamente en GitHub y puede ser revisado por cualquier persona.

La aplicación no cuenta actualmente con firma de código, por lo que tu sistema operativo muestra advertencias de seguridad. Esto no significa que la aplicación sea peligrosa — simplemente significa que el editor no puede ser verificado electrónicamente. La firma de código está planificada para el futuro.
