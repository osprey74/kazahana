kazahana

Ligero. Rápido.
Una mejor forma de usar Bluesky.

kazahana es un cliente de escritorio ligero para Bluesky.
Construido con tecnología nativa del sistema operativo para un uso mínimo de memoria.
Tu experiencia en Bluesky, simplificada.

Características

Cuatro razones por las que kazahana mejora tu experiencia diaria en Bluesky

memory

Ligero

Utiliza el WebView nativo de tu sistema operativo, reduciendo drásticamente el uso de memoria en comparación con PWA. Tu PC sigue rápido, incluso tras horas de uso.

bolt

Rápido

Velocidad de inicio de una aplicación nativa. Carga fluida de la línea de tiempo sin esperas — navega por tu feed sin fricciones.

dashboard

Simple

Una interfaz limpia de una sola columna. Línea de tiempo, publicaciones, notificaciones, búsqueda y perfil — todo lo que necesitas, sin nada de más.

translate

Multilingüe

Compatible con 11 idiomas, incluyendo español y japonés. Los usuarios de Bluesky de todo el mundo pueden usar kazahana en su idioma nativo.

Funcionalidades

Todo lo que puedes hacer con kazahana

dynamic_feed Timeline con actualización automática (intervalo configurable)
bookmark Marcador de posición de lectura
edit_square Publicar y responder
favorite Me gusta y repostear
image Adjuntar imágenes (hasta 4, con texto ALT)
zoom_in Lightbox de imágenes
link Tarjeta de enlace (vista previa OGP)
forum Vista de hilo
notifications Notificaciones (me gusta, repost, seguir, mención, respuesta, cita)
person Perfil y seguir / dejar de seguir
thumb_up Publicaciones con me gusta
photo_library Publicaciones multimedia
search Búsqueda de publicaciones y usuarios
dark_mode Tema oscuro / claro / sistema
translate 11 idiomas
shield Moderación de contenido (etiquetas, desenfoque, ajustes)
tab Feeds personalizados y feeds de listas
lock Threadgate y postgate (restricciones de respuesta/cita)
delete Eliminación de publicaciones
refresh Recarga manual (clic en pestaña / F5 / botón)
vertical_align_top Bandeja del sistema
