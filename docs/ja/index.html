kazahana は軽量なBlueskyデスクトップクライアントです。
OSネイティブの技術で、メモリ消費を最小限に。
あなたのBluesky体験を、もっとシンプルに。

特徴

日常のBluesky体験をもっと快適にする4つの特徴

memory

軽量

OS標準のWebViewを活用し、メモリ消費をPWA比で大幅削減。長時間の利用でもPCが重くなりません。

bolt

高速

ネイティブアプリならではの起動速度。タイムラインの読み込みもスムーズで、待ち時間なく快適に閲覧できます。

dashboard

シンプル

1カラムのすっきりしたUI。タイムライン、投稿、通知、検索、プロフィール — 必要な機能はすべて揃っています。

translate

多言語対応

日本語・英語をはじめ11言語をサポート。世界中のBlueskyユーザーが母国語で使えます。

できること

kazahana で利用できる機能の一覧です

dynamic_feed タイムライン自動更新（間隔設定可能）
bookmark 既読位置マーカー
edit_square 投稿・返信
favorite いいね・リポスト
image 画像添付（最大4枚、ALTテキスト対応）
zoom_in 画像ライトボックス
link リンクカード（OGPプレビュー）
forum スレッド表示
notifications 通知（いいね、リポスト、フォロー、メンション、返信、引用）
person プロフィール・フォロー / フォロー解除
thumb_up いいねした投稿一覧
photo_library メディア投稿一覧
search 投稿・ユーザー検索
dark_mode ダーク / ライト / システムテーマ
translate 11言語対応
shield コンテンツモデレーション（ラベル判定、ブラー、設定）
tab カスタムフィード・リストフィード
lock スレッドゲート・ポストゲート（返信/引用制限）
format_quote 引用投稿
bookmarks ブックマーク
delete 投稿削除
refresh 手動リロード（タブクリック / F5 / ボタン）
vertical_align_top システムトレイ常駐
