# Release Notes — v2.2.7

## EN

### 🔧 Improvements

- **Reorganized post three-dot menu** — Menu items are now grouped by category with separators: general actions (copy link, translate), post moderation (hide, report post, mute thread), and user moderation (mute, block, report user).
- **Reorganized profile three-dot menu** — Added separator between list management and user moderation actions (mute, block, report).
- **Unified menu labels** — Profile menu now uses the same wording as the post menu (e.g. "Mute this user" instead of "Mute") for consistency.

---

## JA

### 🔧 改善

- **投稿の三点メニューを再構成** — メニュー項目をカテゴリごとに区切り線で整理: 一般操作（リンクをコピー、翻訳）、投稿モデレーション（非表示、投稿通報、スレッドミュート）、ユーザーモデレーション（ミュート、ブロック、ユーザー通報）。
- **プロフィールの三点メニューを再構成** — リスト管理とユーザーモデレーション操作（ミュート、ブロック、通報）の間に区切り線を追加。
- **メニュー表記の統一** — プロフィール画面のメニュー表記を投稿メニューと統一（例:「ミュートする」→「このユーザーをミュート」）。
