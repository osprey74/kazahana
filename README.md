# かざはな（Kazahana）

A lightweight Bluesky desktop client built with Tauri v2 + React + TypeScript.

## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run tauri dev

# Build for production
npm run tauri build
```

## Tech Stack

- **Framework**: Tauri v2
- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite
- **Styling**: TailwindCSS
