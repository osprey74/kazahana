# Release Notes — v2.2.6

## EN

### ✨ New Features

- **Block / Unblock user from post menu** — You can now block or unblock a user directly from the three-dot menu on any post in the timeline or thread view. A confirmation dialog is shown before the action is executed.
- **Report user from post menu** — You can now report a user directly from the three-dot menu on any post, in addition to the existing "Report this post" option.

### 📝 Documentation

- Updated PLATFORM_MATRIX: Android inventory — marked 14 implemented features as complete, cleaned up the Desktop → Mobile gap summary
- Updated PLATFORM_MATRIX: Removed profile feeds/lists tabs (iOS), simplified notes

## JA

### ✨ 新機能

- **投稿メニューからユーザーをブロック／ブロック解除** — タイムラインやスレッド表示の投稿メニュー（三点メニュー）から、直接ユーザーをブロック／ブロック解除できるようになりました。実行前に確認ダイアログが表示されます。
- **投稿メニューからユーザーを通報** — 既存の「この投稿を通報」に加え、投稿メニューから直接ユーザーを通報できるようになりました。
