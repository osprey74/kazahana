# Release Notes — v2.2.5

## ✨ New Features / 新機能

- **DM recipient history**: When composing a new direct message, recently contacted users are shown for quick selection.
- **DM 送信先履歴**: 新しいダイレクトメッセージ作成時、最近やりとりしたユーザーをすばやく選択できるようになりました。

## 🐛 Bug Fixes / バグ修正

- **Profile tab reorder**: Changed the profile content tab order to Posts → Replies → Media → Likes → Bookmarks → Starter Packs for a more intuitive layout.
- **プロフィールタブ並び順変更**: プロフィール画面のコンテンツタブを「投稿 → 返信 → メディア → いいね → ブックマーク → スターターパック」に並べ替え、より直感的な配置にしました。

## 📝 Documentation / ドキュメント

- Renamed "Pinned" tab to "Bookmarks" in user guide (EN/JA).
- Added cross-platform feature matrix (`PLATFORM_MATRIX.md`) for tracking Desktop / iOS / Android parity.
- ユーザーガイドのプロフィールタブ名を「付箋」から「ブックマーク」に変更（EN/JA）。
- クロスプラットフォーム機能対応表（`PLATFORM_MATRIX.md`）を追加し、Desktop / iOS / Android の機能差異を管理。
