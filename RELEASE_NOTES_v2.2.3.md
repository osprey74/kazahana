# Release Notes — kazahana v2.2.3

## English

### New Feature

- **Bot Automation Label Badge** — Bluesky's new automation label (self-applied `bot` label) is now supported. A robot icon appears next to the display name of bot accounts across:
  - Post cards (timeline)
  - Profile header
  - User lists (followers, following)
  - Quote embeds
  - Notification items
  - Clicking the icon shows a tooltip explaining the account is automated
  - Supported in all 11 languages

---

## 日本語

### 新機能

- **Bot 自動化ラベルバッジ** — Bluesky の新しい自動化ラベル（セルフラベル `bot`）に対応しました。Bot アカウントの表示名横にロボットアイコンが表示されます：
  - 投稿カード（タイムライン）
  - プロフィールヘッダー
  - ユーザーリスト（フォロワー・フォロー中）
  - 引用埋め込み
  - 通知欄
  - アイコンクリックで「このアカウントは自動化されています」とツールチップ表示
  - 全11言語対応
