# Release Notes — v2.4.3

## EN

### New Features

- **Random watermark placement**: Place a single watermark at a random position within the image bounds
- **Tiling watermark placement**: Checkerboard pattern covering ~20% of the image with rotated (-30°) watermarks; center tile is always fully visible
- **Localized watermark stamp text**: Watermark text burned into images is now translated across all 11 supported languages (previously hardcoded in Japanese)

### Improvements

- **Full i18n coverage for all 11 languages**: Added ~55 missing translation keys across 9 non-EN/JA locales (de, es, fr, id, ko, pt, ru, zh-CN, zh-TW) covering settings, notifications, posts, drafts, and watermark sections

---

## JA

### 新機能

- **ウォーターマークのランダム配置**: 画像内のランダムな位置にウォーターマークを1つ配置
- **ウォーターマークのタイリング配置**: 被覆率約20%の市松模様で画像全体に-30°回転したウォーターマークを繰り返し配置（中央タイルは常に完全に表示）
- **ウォーターマーク焼き込み文言の多言語対応**: 画像に合成される文言がアプリの言語設定に応じて11言語で表示されるようになりました（以前は日本語固定）

### 改善

- **全11言語の翻訳完全対応**: 日本語・英語以外の9言語（独・西・仏・印尼・韓・葡・露・簡中・繁中）に約55個の欠けていた翻訳キーを追加（設定、通知、投稿、下書き、ウォーターマーク各セクション）
